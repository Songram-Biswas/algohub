    Two Sum

    Easy Hash Map Array
    LeetCode ↗

    Given an array of integers nums and an integer target, return the indices of the two numbers that add up to target.

    You may assume exactly one solution exists, and you may not use the same element twice.

    Input: nums = [2,7,11,15], target = 9

    Output: [0,1] // nums[0] + nums[1] == 9

    A brute-force check of every pair costs O(n²). Instead, walk the array once. At each index, ask: "have I already seen the number that completes this pair?"

    1. For the current number, compute complement = target − nums[i].
    2. Look up complement in the hash map.
    3. If found, return the stored index and the current index.
    4. Otherwise, store nums[i] → i and continue.
    function twoSum(nums, target) {
  const seen = new Map();

  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];

    if (seen.has(complement)) {
      return [seen.get(complement), i];
    }
    seen.set(nums[i], i);
  }
  return []; // no solution found
}

    Time

    O(n)

    Single pass through the array.

    Space

    O(n)

    Hash map stores up to n entries.

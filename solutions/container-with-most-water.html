11. Container With Most Water

Medium
Solve on LeetCode

Description

You are given an integer array height, where each value represents the height of a vertical line. Pick two lines so that, together with the x-axis, they form a container that holds the maximum water. Return the maximum area.

Example 1

Input: height = [2,7,4,8,3,6]

Output: 24

Explanation: Using lines at indices 1 and 5, width is 4 and height is min(7,6)=6, so area is 4 * 6 = 24.

Example 2

Input: height = [2,4,7,8,3,10,6,4]

Output: 24

Explanation: Using lines at indices 2 and 6, width is 4 and height is min(7,6)=6, so area is 4 * 6 = 24.

Constraints:

  • n == height.length
  • 2 <= n <= 10⁵
  • 0 <= height[i] <= 10⁴

Idea & Intuition

Use two pointers: left at the start and right at the end. At each step compute current area: (right - left) * min(height[left], height[right]). Move the pointer with the smaller height, because the smaller side limits the area.

flowchart TD A["Start: left=0, right=n-1, max_area=0"] --> B{"left < right?"} B -- No --> C["Return max_area"] B -- Yes --> D["width = right - left"] D --> E["h = min(height[left], height[right])"] E --> F["area = width * h"] F --> G["max_area = max(max_area, area)"] G --> H{"height[left] < height[right]?"} H -- Yes --> I["left += 1"] H -- No --> J["right -= 1"] I --> B J --> B

Python Code

from typing import List

class Solution:
    def maxArea(self, height: List[int]) -> int:
        left = 0
        right = len(height) - 1
        max_area = 0

        while left < right:
            width = right - left
            h = min(height[left], height[right])
            area = width * h
            max_area = max(max_area, area)

            if height[left] < height[right]:
                left += 1
            else:
                right -= 1

        return max_area

Time Complexity: O(n) - Each pointer moves inward at most n times total.

Space Complexity: O(1) - We only keep a few constant-size variables.
